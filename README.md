# @kontourai/datum

The Kontour portfolio's AI provider/model/role registry. Define providers,
models, and roles once; resolve *which backend, which model, whose key, what
base URL* everywhere; generate the native config every tool wants. Datum
abstracts configuration resolution, not invocation — it makes no model calls.

Slice 1 lands the registry, resolver, CLI, and opencode generator. See the PR.
